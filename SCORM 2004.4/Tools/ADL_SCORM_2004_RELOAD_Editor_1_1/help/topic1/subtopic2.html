What Does RELOAD Do?
RELOAD Editor is a Content Package and Metadata Editor. With RELOAD Editor, you can create, import, edit, and export Content Packages.

RELOAD Editor is under active development at the moment.

  • For more information on Metadata, refer to the appropriate section of the Help files: Metadata Resources
