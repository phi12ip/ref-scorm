Contribute
As RELOAD is Open Source, anyone who wants to can contribute to the project. At present (whilst the software is still in Beta) we are not using the CVS facilities of SourceForge, but as the software becomes more stable the code will be available from: http://www.sourceforge.net/projects/x4l-reload/. We are currently preparing some developer documentation which will be of use to anyone wishing to integrate their own code. The developer documentation will be distributed through the RELOAD sourceforge pagesas well. In the meantime, you may wish to look at the To Do list on the RELOAD web site to see what needs to be done.

Other ways to contribute
Even if you can't program, you can contribute to the project by using the tools and providing feedback in the form of ideas, requests, bugs, anomalies and examples. If you have anthing which you feel may be useful to the community as a whole then please let us know. Send an email to Colin Milligan at colin.mlligan@strath.ac.uk
