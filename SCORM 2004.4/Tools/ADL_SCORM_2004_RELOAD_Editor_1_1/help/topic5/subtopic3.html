Your First Content Package
The following pages take you through the entire process of creating a content package. The steps involved can be summarised as follows:

  • Collect all resources together.
  • Open RELOAD tool and set up work space.
  • Add Metadata placeholder.
  • Add Organisations and Items.
  • View Package.
  • Restructure, Tidy node names etc.
  • Add Metadata.
  • Save Content Package.
