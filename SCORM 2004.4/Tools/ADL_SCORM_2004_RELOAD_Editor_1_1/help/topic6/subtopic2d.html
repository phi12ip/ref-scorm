View Menu
The View Menu contains specific options to allow the user to view their content package.

View File
The View File Menu Item (View, View File) allows the user to view any file within the package in a web browser.

Preview Content Package
The Preview Content Package Menu Item (View, Preview Content Package) allows the user to preview the whole content package in a web browser.

Status Window
The Status Window Menu Item (View, Status Window) allows the user to choose whether to display or hide the Java status window. The Java status window is where any information Java messages are displayed and these may be useful if you discover a bug in the software.
